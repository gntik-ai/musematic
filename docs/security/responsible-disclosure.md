# Responsible Disclosure

Report security issues to `security@musematic.ai`. Include affected component, reproduction steps, impact, logs or request IDs, and whether the issue is being actively exploited.

Expected response:

| Step | Target |
| --- | --- |
| Initial acknowledgement | Within 24 business hours. |
| Triage and severity | As soon as enough information is available. |
| Remediation plan | Within 5 business days for validated issues. |
| Coordinated disclosure | Agreed with the reporter after a fix or mitigation is available. |

Do not access data that is not yours, disrupt production systems, or disclose the issue publicly before coordination. Operators still need to publish a PGP key and `security.txt` at `https://musematic.ai/.well-known/security.txt`.
