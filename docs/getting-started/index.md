Start here if you are new to Musematic. This section explains what the platform does, how to run it locally, the vocabulary used across the docs, and a first tutorial that connects agent registration, execution, and observability.

Recommended path:

1. Read [What Is Musematic?](what-is-musematic.md).
2. Run the [Quick Start](quick-start.md).
3. Keep the [Glossary](glossary.md) nearby while reading the rest of the docs.
4. Complete the [First Tutorial](first-tutorial.md) before building production workflows.
