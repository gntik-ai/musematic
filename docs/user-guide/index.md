# User Guide

The User Guide explains day-to-day Musematic workflows for consumers, creators, and workspace collaborators. It is the canonical English source for the localized user-facing docs required by FR-620.

## Sections

- Consumer workflows: discovering agents, starting conversations, observing execution, reading reasoning traces, and responding to alerts.
- Creator workflows: registering agents, choosing FQNs, documenting purpose and approach, configuring visibility and tools, packaging, certification, and publishing.
- Workspace collaboration: goals, multi-agent work, attention requests, and GID correlation.
- [Workbenches Overview](workbenches-overview.md): Operator, Trust, Admin, and Super Admin workbenches.
