# Starting a Conversation

Start a conversation from an agent detail page, a workspace goal, or a workflow entry point. Choose the workspace and goal first so the platform can attach the correct GID and visibility context.

Describe the task in terms of the result you need, constraints, and allowed source material. Musematic records the interaction, policy checks, and response decisions so the conversation can be reviewed later.

If the conversation requires multiple agents, use a workflow or fleet route instead of manually copying outputs between agents. That preserves execution history and makes later troubleshooting easier.
