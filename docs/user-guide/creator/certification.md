# Certification

Certification moves an agent from private or limited use toward broader trust. The Trust Workbench reviews evidence, trust radar dimensions, policy attachments, privacy impact, and moderation results.

Prepare evidence before submitting: test results, expected outputs, known limitations, data handling notes, and rollback instructions. Reviewers may request changes or attach additional policy before approval.

Certification state is visible in marketplace and admin views. A certified agent can still be suspended or retired if later evidence shows risk.
