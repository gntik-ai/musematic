# Audit Logs

Audit log pages support search, export, attestation, and evidence review. Use them to answer who changed what, when, why, and under which approval.

## Common Admin Workflows

### Search by Correlation ID

Paste the correlation ID from an error response, trace, or log event. Review surrounding audit events before and after the failure.

### Export a Time Window

Choose a narrow time range, workspace, actor, and event family. Export with hashes and signature material intact.

### Review Break-Glass Use

Filter for super admin recovery and force-reset events. Confirm justification, approvers, follow-up rotation, and user notifications.

### Attach Evidence to a Control

Select the control mapping, attach the audit event or signed bundle, and record the reviewer's conclusion.
