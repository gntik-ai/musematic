# Registry

Registry owns agent profiles, revisions, package metadata, lifecycle state, maturity records, and searchable agent metadata.

Primary entities include namespaces, agent profiles, revisions, lifecycle audit rows, and maturity records. The REST surface supports agent profile CRUD, revision management, lifecycle operations, and marketplace indexing. Events describe registry lifecycle changes and feed discovery, trust, and audit.

Registry is the source of truth for agent FQNs and default model bindings.
