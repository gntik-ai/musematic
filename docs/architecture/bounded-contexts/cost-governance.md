# Cost Governance

Cost Governance owns cost attribution, workspace budgets, alerting, forecasts, anomalies, hard caps, overrides, and chargeback.

Primary entities include cost attributions, workspace budgets, budget alerts, forecasts, anomalies, and chargeback reports. REST APIs expose budget management and reports. Redis hot counters enforce period spend and short-lived overrides.

Cost Governance uses ClickHouse for cost events and analytics rollups.
