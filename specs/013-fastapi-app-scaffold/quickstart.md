# Quickstart: FastAPI Application Scaffold

**Feature**: 013-fastapi-app-scaffold  
**Path**: `apps/control-plane/`

---

## Prerequisites

```bash
# Python 3.12+
python3 --version  # 3.12.x

# Install dependencies
cd apps/control-plane
pip install -e ".[dev]"

# Local dependencies (Docker Compose)
cd deploy/local && docker compose up -d postgres redis kafka qdrant neo4j clickhouse opensearch minio
```

---

## Run API Profile

```bash
cd apps/control-plane

# Set required env vars
export POSTGRES_DSN="postgresql+asyncpg://musematic:musematic@localhost:5432/musematic"
export REDIS_URL="redis://localhost:6379"
export REDIS_TEST_MODE="standalone"
export KAFKA_BROKERS="localhost:9092"
export PLATFORM_PROFILE="api"

# Start
python -m entrypoints.api_main
# or: uvicorn src.platform.main:create_app --factory --host 0.0.0.0 --port 8000
```

## Test: Correlation ID Propagation

```bash
# Without header — auto-generated
curl -v http://localhost:8000/health 2>&1 | grep -i x-correlation-id
# Expected: X-Correlation-ID: <auto-generated-uuid>

# With header — propagated
curl -v -H "X-Correlation-ID: test-123" http://localhost:8000/health 2>&1 | grep -i x-correlation-id
# Expected: X-Correlation-ID: test-123
```

## Test: Exception Hierarchy

```python
from src.platform.common.exceptions import NotFoundError, BudgetExceededError

try:
    raise NotFoundError("AGENT_NOT_FOUND", "Agent not found", {"agent_id": "test"})
except NotFoundError as e:
    assert e.status_code == 404
    assert e.code == "AGENT_NOT_FOUND"

try:
    raise BudgetExceededError("BUDGET_EXCEEDED", "Token limit reached")
except BudgetExceededError as e:
    assert e.status_code == 429
```
